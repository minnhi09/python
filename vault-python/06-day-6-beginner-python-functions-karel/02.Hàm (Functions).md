### Khái niệm cơ bản

Hàm (function) là một khối mã lập trình được đặt tên, có thể thực hiện một chức năng cụ thể. Giống như con người có thể đảm nhận nhiều vai trò khác nhau (mẹ, cha, con trai, con gái), code của chúng ta cũng có thể thực hiện nhiều chức năng khác nhau thông qua các hàm.

Trong Python, chúng ta đã sử dụng nhiều hàm mà không nhận ra. Bạn có thể nhận biết một hàm qua cú pháp: **tên hàm** theo sau bởi **cặp dấu ngoặc đơn** `()`.

### Built-in Functions có sẵn

Python cung cấp sẵn nhiều hàm tích hợp mà chúng ta đã sử dụng:

- `len()` - Đếm số phần tử trong một tập hợp
- `int()` - Chuyển đổi giá trị thành số nguyên (integer)
- `print()` - In kết quả ra màn hình
- `range()` - Tạo dãy số

Ví dụ sử dụng:

```python
print("Hello")  # In ra "Hello"

num_char = len("Hello")  # Đếm số ký tự
print(num_char)  # Kết quả: 5
```


### Tạo hàm riêng (Custom Functions)

#### Cú pháp định nghĩa hàm

Để tạo hàm riêng, chúng ta sử dụng từ khóa `def` (viết tắt của define):

```python
def my_function():
    print("Hello")
    print("Bye")
```

Cấu trúc gồm:

- Từ khóa `def`
- Tên hàm (ví dụ: `my_function`)
- Cặp ngoặc đơn `()`
- Dấu hai chấm `:`
- Các dòng code bên trong **phải được thụt lề** (indented)


#### Gọi hàm (Calling a Function)

Sau khi định nghĩa, chúng ta cần **gọi** (call) hàm để thực thi nó:

```python
my_function()  # Thực thi hàm
```

**Lưu ý quan trọng:** Chỉ định nghĩa hàm thôi chưa đủ - hàm sẽ không chạy cho đến khi bạn gọi nó.

### Cách hoạt động của hàm

Khi chương trình gặp lệnh gọi hàm:

1. Tìm kiếm định nghĩa của hàm đó
2. Nhảy đến vị trí định nghĩa hàm
3. Thực thi từng dòng code bên trong hàm (từ trên xuống dưới)
4. Quay lại vị trí ban đầu và tiếp tục chạy code

### Thực hành với Reeborg's World

Reeborg's World là một công cụ học tập tương tác giống như Karel the Robot, cho phép lập trình bằng Python để điều khiển một robot.

#### Các hàm cơ bản

- `move()` - Di chuyển robot tiến một bước
- `turn_left()` - Quay robot sang trái

Ví dụ di chuyển robot 3 bước:

```python
move()
move()
move()
```


#### Tạo hàm tùy chỉnh

Vấn đề: Reeborg's World chỉ có `turn_left()`, không có `turn_right()` hoặc `turn_around()`.

Giải pháp - Tạo hàm `turn_around()`:

```python
def turn_around():
    turn_left()
    turn_left()
```

Sử dụng hàm mới:

```python
move()
move()
turn_around()  # Thay vì viết turn_left() hai lần
move()
move()
```


#### Tạo hàm `turn_right()`

```python
def turn_right():
    turn_left()
    turn_left()
    turn_left()
```

Quay trái 3 lần = quay phải 1 lần.

### Ví dụ thực tế: Vẽ hình vuông

Sử dụng hàm `turn_right()` để vẽ hình vuông:

```python
def turn_right():
    turn_left()
    turn_left()
    turn_left()

turn_left()
move()
turn_right()
move()
turn_right()
move()
turn_right()
move()
```


### Lợi ích của Functions

- **Giảm số dòng code:** Thay vì viết `turn_left()` ba lần mỗi khi cần quay phải, chỉ cần gọi `turn_right()`
- **Code dễ đọc hơn:** `turn_right()` rõ ràng hơn nhiều so với ba dòng `turn_left()`
- **Tái sử dụng:** Định nghĩa một lần, sử dụng nhiều lần
- **Dễ bảo trì:** Nếu cần thay đổi logic, chỉ sửa ở một nơi (trong định nghĩa hàm)


### Debug với Step-through

Reeborg's World có nút **step-through** (⏯️) cho phép thực thi từng dòng code một, giúp:

- Quan sát cách mỗi lệnh được thực hiện
- Hiểu luồng thực thi của chương trình
- Tìm lỗi dễ dàng hơn

Khi gặp lệnh gọi hàm, chương trình sẽ:

1. Nhảy đến định nghĩa hàm
2. Thực hiện từng dòng trong hàm
3. Quay lại vị trí gọi hàm và tiếp tục
