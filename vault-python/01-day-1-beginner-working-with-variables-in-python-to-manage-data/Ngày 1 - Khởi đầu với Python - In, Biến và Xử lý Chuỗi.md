
### Tổng quan bài học

Ngày đầu tiên của hành trình 100 ngày học Python sẽ tập trung vào các khái niệm cơ bản sau:

- In dữ liệu ra màn hình (Printing)
- Chú thích code (Commenting)
- Gỡ lỗi (Debugging)
- Xử lý chuỗi ký tự (String Manipulation)
- Biến (Variables)


### Dự án: Band Name Generator (Trình Tạo Tên Ban Nhạc)

**Mục tiêu:** Xây dựng chương trình giúp tạo tên cho ban nhạc dựa trên thông tin người dùng nhập vào.

**Cách hoạt động:**

- Chương trình hỏi tên thành phố nơi bạn lớn lên
- Chương trình hỏi tên thú cưng của bạn
- Kết hợp hai thông tin để tạo tên ban nhạc

**Ví dụ minh họa:**

```
Nhập tên thành phố: Bristol
Nhập tên thú cưng: Rabbit
→ Tên ban nhạc của bạn: Bristol Rabbit
```


### Các khái niệm sẽ học

Để hoàn thành dự án này, bạn sẽ làm quen với:

- Cách in dữ liệu (Printing)
- Nhập dữ liệu từ người dùng (Inputting)
- Viết chú thích trong code (Commenting)
- Xử lý các lỗi thường gặp:
    - Lỗi tên biến (Name Errors)
    - Lỗi cú pháp (Syntax Errors)
- Thao tác với chuỗi ký tự (String Manipulation)
- Sử dụng biến để lưu trữ dữ liệu (Variables)
