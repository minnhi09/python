



### Tại Sao Cần Tính Ngẫu Nhiên?

Tính ngẫu nhiên (randomization) rất quan trọng khi muốn tạo chương trình có tính bất định (unpredictability). Ứng dụng phổ biến nhất là trong game development.

**Ví dụ thực tế:**

- Game Tetris: Nếu các khối rơi xuống theo thứ tự cố định → Không thú vị
- Cần tính ngẫu nhiên để tạo trải nghiệm đa dạng và hấp dẫn

**Thách thức:**

- Máy tính là thiết bị định trước (deterministic) → Hoạt động theo cách có thể dự đoán
- Làm sao để tạo tính ngẫu nhiên từ máy móc hoạt động bằng 1 và 0?

Python sử dụng thuật toán **Mersenne Twister** để tạo số ngẫu nhiên giả (pseudo-random numbers).

### Sử dụng Random Module

#### Import Module

Python đã tạo sẵn Random Module với nhiều hàm hữu ích. Không cần tự viết thuật toán phức tạp.

```python
import random
```


#### Tạo Số Nguyên Ngẫu Nhiên (Random Integer)

Sử dụng hàm `random.randint(a, b)` để tạo số nguyên ngẫu nhiên trong khoảng từ a đến b (bao gồm cả a và b).

```python
import random

# Tạo số ngẫu nhiên từ 1 đến 10 (bao gồm 1 và 10)
random_integer = random.randint(1, 10)
print(random_integer)
```

**Cú pháp:**

- `random` → Tên module đã import
- `.` (dấu chấm/dot) → Truy cập vào các hàm bên trong module
- `randint(a, b)` → Hàm tạo số ngẫu nhiên
    - `a`: Giá trị nhỏ nhất (inclusive)
    - `b`: Giá trị lớn nhất (inclusive)


### Module Là Gì?

Module là cách tổ chức code thành các file riêng biệt, mỗi file chịu trách nhiệm cho một chức năng cụ thể.

**Lợi ích:**

- Chia nhỏ dự án phức tạp thành các phần dễ quản lý
- Nhiều người có thể làm việc cùng lúc trên các module khác nhau
- Dễ bảo trì và tái sử dụng code


#### Tạo Module Riêng

**Bước 1:** Tạo file mới `my_module.py`

```python
# my_module.py
my_favorite_number = 3.1415
```

**Bước 2:** Import và sử dụng trong file chính

```python
# task.py hoặc main.py
import my_module

print(my_module.my_favorite_number)  # Output: 3.1415
```


### Tạo Số Thập Phân Ngẫu Nhiên (Random Float)

#### Hàm random.random()

Tạo số thập phân ngẫu nhiên từ 0 (inclusive) đến 1 (exclusive).

```python
import random

# Số ngẫu nhiên từ 0 đến 1 (không bao gồm 1)
random_number_0_to_1 = random.random()
print(random_number_0_to_1)
```

**Mở rộng khoảng giá trị:**

```python
# Số ngẫu nhiên từ 0 đến 10 (không bao gồm 10)
random_number_0_to_10 = random.random() * 10
print(random_number_0_to_10)
```

**Lưu ý cú pháp:**

- `random.random()` → `random` đầu tiên là tên module, `random()` thứ hai là tên hàm
- Hàm không nhận tham số đầu vào nhưng vẫn cần dấu ngoặc đơn `()`


#### Hàm random.uniform(a, b)

Tạo số thập phân ngẫu nhiên trong khoảng từ a đến b (bao gồm cả a và b, tùy thuộc vào làm tròn).

```python
import random

# Số thập phân ngẫu nhiên từ 1 đến 10
random_float = random.uniform(1, 10)
print(random_float)
```

**So sánh:**

- `random.random()`: Khoảng semi-open [0, 1) → Đáng tin cậy hơn khi mở rộng
- `random.uniform(a, b)`: Có thể không chắc chắn về giới hạn trên do làm tròn


### Bài Tập Thực Hành: Chương Trình Toss Coin

**Yêu cầu:** Tạo chương trình tung đồng xu in ra "Heads" hoặc "Tails" ngẫu nhiên.

**Giải pháp:**

```python
import random

# Tạo số ngẫu nhiên 0 hoặc 1
random_heads_or_tails = random.randint(0, 1)

# Kiểm tra và in kết quả
if random_heads_or_tails == 0:
    print("Heads")
else:
    print("Tails")
```

**Logic:**

- Tạo số ngẫu nhiên: 0 hoặc 1
- 0 → "Heads"
- 1 → "Tails"
- Xác suất 50/50 cho mỗi kết quả


### Ghi Chú Quan Trọng

**Về hàm (Functions):**

- Một số hàm cần tham số đầu vào: `randint(1, 10)`, `print("Hello")`
- Một số hàm không cần tham số: `random.random()` → Vẫn cần dấu ngoặc đơn để kích hoạt

**Về Module:**

- Module giống như các bộ phận của ô tô (bánh xe, khung xe, động cơ)
- Mỗi module thực hiện một chức năng riêng
- Ghép các module lại → Chương trình hoàn chỉnh
