### Danh Sách Là Gì?

**List** (danh sách) là một cấu trúc dữ liệu (Data Structure) quan trọng trong Python, được sử dụng để tổ chức và lưu trữ nhiều giá trị cùng lúc.

**So sánh với biến thông thường:**

- Biến đơn: `a = 3` hoặc `b = "hello"` → Chỉ lưu 1 giá trị
- List: Lưu nhiều giá trị có liên quan với nhau trong một biến duy nhất

**Khi nào cần dùng List:**

- Lưu trữ dữ liệu có mối liên hệ (ví dụ: tên các bang của Mỹ)
- Duy trì thứ tự dữ liệu (ví dụ: hàng đợi khách hàng)
- Quản lý tập hợp dữ liệu động có thể thay đổi


### Cú Pháp Tạo List

List được định nghĩa bằng cặp ngoặc vuông `[]` với các phần tử cách nhau bởi dấu phẩy.

```python
# Cấu trúc cơ bản
list_name = [item1, item2, item3]

# Ví dụ: Danh sách trái cây
fruits = ["Cherry", "Apple", "Pear"]

# List có thể chứa nhiều kiểu dữ liệu khác nhau
mixed_list = ["Hello", 123, True, 3.14]
```

**Ví dụ thực tế: Danh sách các bang Mỹ**

```python
# Cách cũ (không hiệu quả)
state1 = "Delaware"
state2 = "Pennsylvania"
state3 = "New Jersey"

# Cách mới sử dụng List
states_of_america = ["Delaware", "Pennsylvania", "New Jersey"]
```


### Chỉ Mục List (List Indexing)

List trong Python sử dụng hệ thống đánh số bắt đầu từ **0** (zero-based indexing).

**Chỉ mục dương (Positive Index):**

- `[0]` → Phần tử đầu tiên
- `[1]` → Phần tử thứ hai
- `[2]` → Phần tử thứ ba

```python
states_of_america = ["Delaware", "Pennsylvania", "New Jersey"]

print(states_of_america[0])  # Delaware
print(states_of_america[1])  # Pennsylvania
print(states_of_america[2])  # New Jersey
```

**Tại sao bắt đầu từ 0?**

Hãy nghĩ về chỉ mục như là độ dịch chuyển (offset) từ điểm bắt đầu của list:

- `[0]` → Không dịch chuyển (ngay điểm đầu)
- `[1]` → Dịch 1 vị trí từ đầu
- `[2]` → Dịch 2 vị trí từ đầu

**Chỉ mục âm (Negative Index):**

Đếm ngược từ cuối list:

- `[-1]` → Phần tử cuối cùng
- `[-2]` → Phần tử áp cuối
- `[-3]` → Phần tử thứ 3 từ cuối

```python
states_of_america = ["Delaware", "Pennsylvania", "New Jersey", "Arizona", "Alaska", "Hawaii"]

print(states_of_america[-1])  # Hawaii (cuối cùng)
print(states_of_america[-2])  # Alaska (áp cuối)
```

**Lưu ý:** Không có `-0` vì về mặt toán học `-0 = 0`

### Thay Đổi Phần Tử Trong List

List là cấu trúc dữ liệu có thể thay đổi (mutable). Bạn có thể cập nhật giá trị của bất kỳ phần tử nào.

```python
states_of_america = ["Delaware", "Pennsylvania", "New Jersey"]

# Thay đổi phần tử tại index 1
states_of_america[1] = "Pencilvania"

print(states_of_america)
# Output: ["Delaware", "Pencilvania", "New Jersey"]
```


### Các Phương Thức Thao Tác List

#### Thêm Phần Tử Vào Cuối: append()

Thêm một phần tử vào cuối list.

```python
states_of_america = ["Delaware", "Pennsylvania", "New Jersey"]

# Thêm phần tử mới
states_of_america.append("Angelaland")

print(states_of_america)
# Output: ["Delaware", "Pennsylvania", "New Jersey", "Angelaland"]
```


#### Mở Rộng List: extend()

Thêm nhiều phần tử (từ một list khác) vào cuối list hiện tại.

```python
states_of_america = ["Delaware", "Pennsylvania", "New Jersey"]

# Mở rộng với nhiều phần tử
states_of_america.extend(["California", "Texas"])

print(states_of_america)
# Output: ["Delaware", "Pennsylvania", "New Jersey", "California", "Texas"]
```

**So sánh append() và extend():**

- `append()`: Thêm 1 phần tử → `list.append(item)`
- `extend()`: Thêm nhiều phần tử từ list khác → `list.extend([item1, item2])`


### Các Phương Thức List Khác

Python cung cấp nhiều phương thức hữu ích cho list. Một số ví dụ:

- `insert(index, item)`: Chèn phần tử tại vị trí chỉ định
- `remove(item)`: Xóa phần tử đầu tiên có giá trị tương ứng
- `pop(index)`: Xóa và trả về phần tử tại vị trí chỉ định
- `clear()`: Xóa tất cả phần tử
- `sort()`: Sắp xếp list
- `reverse()`: Đảo ngược thứ tự list

**Không cần học thuộc lòng!** Hãy tham khảo documentation khi cần.

### Tóm Tắt Cú Pháp

```python
# Tạo list
my_list = ["item1", "item2", "item3"]

# Truy cập phần tử
first_item = my_list[0]
last_item = my_list[-1]

# Thay đổi phần tử
my_list[1] = "new_item"

# Thêm phần tử
my_list.append("item4")
my_list.extend(["item5", "item6"])

# In list
print(my_list)
```


### Ví Dụ Tổng Hợp

```python
# Danh sách các bang Mỹ theo thứ tự gia nhập Liên bang
states_of_america = [
    "Delaware",      # 0 (bang đầu tiên)
    "Pennsylvania",  # 1
    "New Jersey",    # 2
    "Georgia",       # 3
    # ... thêm các bang khác
    "Alaska",        # -2
    "Hawaii"         # -1 (bang cuối cùng)
]

# Truy cập
print(states_of_america[0])   # Delaware
print(states_of_america[-1])  # Hawaii

# Sửa đổi
states_of_america[1] = "Penn"

# Thêm mới
states_of_america.append("Puerto Rico")
```


### Ghi Chú Quan Trọng

**Nhận biết List trong code:**

- Thấy `[]` → Có thể liên quan đến list
- Tạo list: `my_list = []`
- Truy cập phần tử: `my_list[index]`

**Thứ tự trong List:**

- List duy trì thứ tự các phần tử
- Thứ tự không bị mất khi lưu vào biến
- Quan trọng khi dữ liệu cần sắp xếp (ví dụ: hàng đợi, timeline)
