### Thử Thách Lập Trình

Viết code Python để chọn ngẫu nhiên một tên từ danh sách bạn bè. Mỗi lần chạy chương trình sẽ in ra một tên khác nhau.

### Hai Phương Pháp Giải Quyết

#### Phương Pháp 1: Sử dụng random.choice()

Đây là cách đơn giản nhất, sử dụng hàm có sẵn trong [[Python Module]] `random`.

```python
import random

friends = ["Alice", "Bob", "Charlie", "David", "Emily"]

# Chọn ngẫu nhiên một phần tử từ list
print(random.choice(friends))
```

**Cách tìm ra giải pháp này:**

- Tìm trong documentation (tài liệu) của module `random`
- Hoặc Google search: "get random item from list Python"
- Hàm `choice()` nhận vào một sequence (chuỗi/dãy) - [[List]] là một loại sequence trong Python


#### Phương Pháp 2: Sử dụng random.randint() với Index

Phương pháp này dựa trên kiến thức đã học về [[Random Number]] và [[List Indexing]].

```python
import random

friends = ["Alice", "Bob", "Charlie", "David", "Emily"]

# Tạo một số ngẫu nhiên từ 0 đến 4 (index của list)
random_index = random.randint(0, 4)

# Sử dụng index để lấy phần tử từ list
print(friends[random_index])
```

**Giải thích:**

- List trong Python đếm từ index 0 (Alice ở vị trí 0)
- `randint(0, 4)` tạo số ngẫu nhiên từ 0 đến 4 (bao gồm cả 0 và 4)
- Sử dụng `friends[random_index]` để truy xuất phần tử tại vị trí ngẫu nhiên


### So Sánh Hai Phương Pháp

- **Phương pháp 1** (`random.choice()`): Ít code hơn, dễ đọc hơn, được ưu tiên nếu bạn biết hàm này tồn tại
- **Phương pháp 2** (`random.randint()`): Sử dụng kiến thức cơ bản đã học, cũng hoàn toàn hợp lệ

Cả hai phương pháp đều cho kết quả giống nhau và đều đúng. Lựa chọn phương pháp nào phụ thuộc vào kiến thức và phong cách lập trình của bạn.
