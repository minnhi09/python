



### Vấn Đề Khởi Đầu

Khi sử dụng hàm `len()` để đếm ký tự trong chuỗi, mọi thứ hoạt động bình thường:

```python
print(len("hello"))  # Kết quả: 5
```

Nhưng nếu thử đếm số chữ số trong một số:

```python
print(len(12345))  # TypeError!
```

Chương trình sẽ báo lỗi `TypeError` liên quan đến kiểu dữ liệu `'int'`. Điều này dẫn đến câu hỏi: Kiểu dữ liệu (Data Types) là gì?

### Các Kiểu Dữ Liệu Cơ Bản

Python có 4 kiểu dữ liệu cơ bản quan trọng:

- **String** (Chuỗi)
- **Integer** (Số nguyên)
- **Float** (Số thập phân)
- **Boolean** (Logic)


### String - Chuỗi Ký Tự

**Đặc điểm:**

- Là chuỗi các ký tự được bao quanh bởi dấu ngoặc kép `""`
- Mỗi ký tự có vị trí riêng (index)

**Subscripting - Truy xuất ký tự:**

```python
print("hello"[0])  # Kết quả: H (ký tự đầu tiên)
print("hello"[4])  # Kết quả: o (ký tự cuối cùng)
```

**Lưu ý quan trọng:** Lập trình viên luôn đếm từ 0, không phải từ 1!

**Mẹo:** Sử dụng chỉ số âm để đếm từ cuối chuỗi:

```python
print("hello"[-1])  # Kết quả: o (ký tự cuối)
print("hello"[-2])  # Kết quả: l (ký tự thứ 2 từ cuối)
```

**Số trong dạng string:**

```python
print("123" + "345")  # Kết quả: 123345 (nối chuỗi, không phải phép cộng)
```

Khi số được đặt trong dấu ngoặc kép, Python xử lý chúng như văn bản, không phải số học.

### Integer - Số Nguyên

**Đặc điểm:**

- Số nguyên không có phần thập phân
- Có thể là số dương hoặc âm

```python
print(123 + 345)  # Kết quả: 468 (phép cộng thực sự)
```

**Định dạng số lớn:** Sử dụng dấu gạch dưới `_` thay cho dấu phẩy để dễ đọc:

```python
123_456_789  # Tương đương: 123456789
```

Python sẽ tự động bỏ qua dấu gạch dưới, chúng chỉ giúp con người đọc dễ hơn.

### Float - Số Thập Phân

**Đặc điểm:**

- Số có phần thập phân
- Tên gọi xuất phát từ "floating point number" (điểm thập phân có thể "trượt" đến bất kỳ vị trí nào)

```python
3.14159  # Đây là float
```


### Boolean - Giá Trị Logic

**Đặc điểm:**

- Chỉ có 2 giá trị: `True` hoặc `False`
- Viết hoa chữ cái đầu
- Không có dấu ngoặc kép

```python
is_active = True
is_complete = False
```

Boolean được sử dụng rất nhiều để kiểm tra điều kiện và điều khiển luồng chương trình.

### So Sánh Các Kiểu Dữ Liệu

| Kiểu | Ví dụ | Đặc điểm |
| :-- | :-- | :-- |
| String | `"hello"` | Văn bản trong ngoặc kép |
| Integer | `123` | Số nguyên |
| Float | `3.14` | Số có phần thập phân |
| Boolean | `True` | Giá trị đúng/sai |

### Ghi Chú Quan Trọng

- Luôn đếm từ 0 trong lập trình
- Số trong ngoặc kép `"123"` khác với số `123`
- Mỗi kiểu dữ liệu có cách xử lý và phép toán riêng
- TypeError xảy ra khi sử dụng sai kiểu dữ liệu
