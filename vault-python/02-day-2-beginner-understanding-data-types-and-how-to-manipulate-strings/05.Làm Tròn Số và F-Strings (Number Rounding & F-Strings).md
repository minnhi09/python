### Vấn Đề Với Số Thập Phân Dài

Khi tính toán BMI hoặc các phép toán khác, kết quả thường có nhiều chữ số thập phân:

```python
bmi = 84 / 1.65 ** 2
print(bmi)  # Kết quả: 30.8547619047619...
```

Đây không phải cách hiển thị lý tưởng cho người dùng. Chúng ta cần làm tròn số!

### Cách Làm Tròn Số

**Chuyển thành Integer (không khuyến khích):**

```python
bmi = 84 / 1.65 ** 2
print(int(bmi))  # Kết quả: 30
```

Phương pháp này gọi là **flooring** (làm tròn xuống) - loại bỏ tất cả chữ số thập phân mà không làm tròn theo quy tắc toán học.

**Sử dụng round() (khuyến khích):**

```python
bmi = 84 / 1.65 ** 2
print(round(bmi))  # Kết quả: 31
```

Hàm `round()` làm tròn theo quy tắc toán học truyền thống:

- Từ 0.5 trở lên → làm tròn lên
- Dưới 0.5 → làm tròn xuống

**So sánh các phương pháp:**

```python
bmi = 30.85

print(bmi)          # 30.85 (gốc)
print(int(bmi))     # 30 (flooring - cắt bỏ)
print(round(bmi))   # 31 (làm tròn toán học)
```


### Làm Tròn Đến Số Chữ Số Cụ Thể

Hàm `round()` nhận 2 tham số:

- **Tham số 1:** Số cần làm tròn
- **Tham số 2:** Số chữ số thập phân (tùy chọn)

```python
bmi = 30.85426

print(round(bmi, 2))  # Kết quả: 30.85
print(round(bmi, 3))  # Kết quả: 30.854
print(round(bmi, 1))  # Kết quả: 30.9
```

**Ứng dụng thực tế:** Rất hữu ích khi làm việc với tiền tệ, chuyển đổi đơn vị, hoặc bất kỳ phép tính nào cần độ chính xác cụ thể.

### Assignment Operators - Toán Tử Gán

Các toán tử gán giúp cập nhật giá trị biến dựa trên giá trị cũ một cách ngắn gọn.

**Phép cộng tích lũy (+=):**

```python
score = 0
score += 1  # Tương đương: score = score + 1
print(score)  # Kết quả: 1
```

**Các toán tử gán khác:**

```python
score = 10

score += 5   # score = 15 (cộng thêm 5)
score -= 3   # score = 12 (trừ đi 3)
score *= 2   # score = 24 (nhân với 2)
score /= 4   # score = 6.0 (chia cho 4)
```

**Bảng tóm tắt:**


| Toán tử | Ý nghĩa | Ví dụ | Tương đương |
| :-- | :-- | :-- | :-- |
| `+=` | Cộng thêm | `x += 3` | `x = x + 3` |
| `-=` | Trừ đi | `x -= 2` | `x = x - 2` |
| `*=` | Nhân với | `x *= 4` | `x = x * 4` |
| `/=` | Chia cho | `x /= 5` | `x = x / 5` |

**Khi nào sử dụng:** Rất hữu ích khi theo dõi điểm số, tổng tiền, số lượng, hoặc bất kỳ giá trị tích lũy nào.

### F-Strings - Định Dạng Chuỗi Linh Hoạt

**Vấn đề với cách truyền thống:**

```python
score = 10
print("Your score is " + score)  # TypeError!
```

Không thể nối chuỗi với số trực tiếp. Phải chuyển đổi kiểu:

```python
print("Your score is " + str(score))  # Hoạt động nhưng phức tạp
```

**Giải pháp: F-Strings**

Thêm ký tự `f` trước chuỗi và đặt biến trong dấu ngoặc nhọn `{}`:

```python
score = 10
print(f"Your score is {score}")  # Your score is 10
```

**Kết hợp nhiều kiểu dữ liệu:**

```python
score = 0
height = 1.8
is_winning = True

print(f"Your score is {score}, your height is {height}, you are winning is {is_winning}")
# Kết quả: Your score is 0, your height is 1.8, you are winning is True
```

**Ưu điểm của F-Strings:**

- Tự động chuyển đổi kiểu dữ liệu
- Dễ đọc và viết
- Không cần dùng toán tử `+` nhiều lần
- Không cần gọi `str()` thủ công

**Cú pháp chi tiết:**

```python
# Cú pháp cơ bản
name = "Alice"
age = 25
print(f"My name is {name} and I am {age} years old")

# Kết hợp với biểu thức
print(f"Next year I will be {age + 1}")

# Kết hợp với hàm
bmi = 30.85426
print(f"Your BMI is {round(bmi, 2)}")
# Kết quả: Your BMI is 30.85
```


### So Sánh Các Cách Định Dạng

**Cách cũ (concatenation):**

```python
score = 100
print("Score: " + str(score) + " points")  # Phức tạp
```

**Cách hiện đại (f-strings):**

```python
score = 100
print(f"Score: {score} points")  # Đơn giản, rõ ràng
```


### Ví Dụ Thực Tế Kết Hợp

```python
# Tính toán BMI và hiển thị kết quả đẹp
weight = 84
height = 1.65

bmi = weight / height ** 2
bmi_rounded = round(bmi, 2)

print(f"Your weight is {weight}kg and height is {height}m")
print(f"Your BMI is {bmi_rounded}")

# Kết quả:
# Your weight is 84kg and height is 1.65m
# Your BMI is 30.85
```


### Ghi Nhớ

**Làm tròn số:**

- `int()` → cắt bỏ phần thập phân (flooring)
- `round()` → làm tròn toán học
- `round(số, n)` → làm tròn đến n chữ số thập phân

**Assignment operators:**

- Viết ngắn gọn hơn khi cập nhật giá trị
- Phổ biến: `+=`, `-=`, `*=`, `/=`

**F-strings:**

- Thêm `f` trước chuỗi
- Đặt biến/biểu thức trong `{}`
- Tự động xử lý chuyển đổi kiểu


### Lưu Ý Quan Trọng

- F-strings chỉ hoạt động từ Python 3.6 trở lên
- Ký tự `f` phải đứng ngay trước dấu ngoặc kép
- Có thể dùng ngoặc kép đơn `'` hoặc kép `"`, nhưng `"` phổ biến hơn
- Flooring và rounding cho kết quả khác nhau - chọn đúng theo ngữ cảnh
