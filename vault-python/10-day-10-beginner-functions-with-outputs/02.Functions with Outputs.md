

## Hàm có Đầu ra (Functions with Outputs)

### Tổng quan 3 loại hàm trong Python

Python có 3 loại hàm chính:

**1. Hàm thông thường (Basic Function)**

- Sử dụng từ khóa `def`, tên hàm, dấu ngoặc đơn rỗng `()`, và dấu hai chấm `:`
- Giúp giảm lượng mã khi có các thao tác cần lặp lại nhiều lần

**2. Hàm có đầu vào (Functions with Inputs)**

- Có tham số (parameters) bên trong dấu ngoặc đơn
- Cho phép hàm thực hiện các hành động khác nhau tùy thuộc vào đối số (arguments) được truyền vào

**3. Hàm có đầu ra (Functions with Outputs)**

- Sử dụng từ khóa `return` để trả về kết quả
- Đầu ra thay thế vị trí gọi hàm trong code


### Cách hoạt động của từ khóa return

Khi hàm thực thi xong, giá trị sau `return` sẽ thay thế vị trí mà hàm được gọi:

```python
def my_function():
    result = 3 * 2
    return result

output = my_function()  # output = 6
```

**Lưu ý quan trọng:** Đầu vào (inputs) và đầu ra (outputs) là hai khái niệm hoàn toàn độc lập. Một hàm có thể có output mà không cần input.

### Ví dụ thực hành: Hàm format_name()

Tạo hàm chuyển đổi tên thành dạng Title Case:

```python
def format_name(f_name, l_name):
    # Chuyển đổi sang Title Case
    formatted_f_name = f_name.title()
    formatted_l_name = l_name.title()
    
    # Trả về chuỗi đã format
    return f"{formatted_f_name} {formatted_l_name}"

# Sử dụng hàm
formatted_string = format_name("aNgElA", "YU")
print(formatted_string)  # Angela Yu
```

**Quy trình hoạt động:**

- Hàm `title()` chuyển chuỗi sang Title Case và trả về kết quả
- Kết quả được lưu vào biến `formatted_f_name` và `formatted_l_name`
- `return` trả về chuỗi đã format hoàn chỉnh
- Giá trị trả về thay thế vị trí gọi hàm `format_name()`


### Cách sử dụng Output

Có nhiều cách để sử dụng output của hàm:

```python
# Cách 1: Lưu vào biến rồi in
formatted_string = format_name("angela", "YU")
print(formatted_string)

# Cách 2: In trực tiếp
print(format_name("angela", "YU"))

# Cách 3: Sử dụng làm input cho hàm khác
result = len(format_name("angela", "YU"))
```


### Sự khác biệt giữa print và return

**Print:**

- Chỉ hiển thị nội dung ra console
- Không trả về giá trị để sử dụng tiếp

**Return:**

- Trả về giá trị có thể được lưu vào biến
- Có thể sử dụng làm input cho hàm khác
- Cho phép kết nối (chain) các hàm với nhau


### Kết nối các hàm (Chaining Functions)

Ví dụ minh họa cách output của hàm này trở thành input của hàm khác:

```python
def function_1(text):
    return text + text

def function_2(text):
    return text.title()

# Output của function_1 → Input của function_2
output = function_1("hello")  # "hellohello"
print(output)

result = function_2(function_1("hello"))
print(result)  # "Hellohello"
```

**Giải thích:**

- `function_1("hello")` trả về `"hellohello"`
- Kết quả này trở thành input cho `function_2()`
- `function_2()` nhận `"hellohello"` và trả về `"Hellohello"`


### Ví dụ với hàm có sẵn

Hàm `len()` cũng hoạt động tương tự - nhận input và trả về output:

```python
name = "Angela"
num_letters = len(name)  # Output: 6
```

Quy trình:

- Input: chuỗi `"Angela"`
- Xử lý: đếm số ký tự
- Output: số `6` thay thế vị trí `len(name)`
- Kết quả được lưu vào `num_letters`


### Ghi chú quan trọng

- Từ khóa `return` là yếu tố then chốt để tạo hàm có output
- Mọi giá trị sau `return` sẽ thay thế vị trí gọi hàm
- Một hàm có thể có nhiều câu lệnh `return` (sẽ học ở bài tiếp theo)
- Có thể kết nối nhiều hàm bằng cách dùng output làm input
