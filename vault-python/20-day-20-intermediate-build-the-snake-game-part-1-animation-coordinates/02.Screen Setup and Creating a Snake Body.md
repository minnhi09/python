

## Tạo Thân Rắn (Snake Body) - Bước 1

### Thiết lập dự án

Tạo project mới tên `snake_game` trong PyCharm với file `main.py`.

**Import các thư viện cần thiết:**

```python
from turtle import Screen, Turtle
```


### Cấu hình màn hình game

Khởi tạo và thiết lập màn hình hiển thị:

```python
screen = Screen()
screen.setup(width=600, height=600)
screen.bgcolor("black")
screen.title("My Snake Game")

screen.exitonclick()
```

**Giải thích:**

- `setup()` - Tạo cửa sổ kích thước 600x600 pixels
- `bgcolor()` - Đặt màu nền đen
- `title()` - Đặt tiêu đề cửa sổ
- `exitonclick()` - Giữ cửa sổ mở cho đến khi click chuột


### Tạo thân rắn bằng 3 segment

#### Cách 1: Tạo từng segment riêng lẻ

```python
# Segment 1 - ở giữa màn hình
segment_1 = Turtle("square")
segment_1.color("white")

# Segment 2 - lùi về bên trái 20 pixels
segment_2 = Turtle("square")
segment_2.color("white")
segment_2.goto(-20, 0)

# Segment 3 - lùi tiếp 20 pixels nữa
segment_3 = Turtle("square")
segment_3.color("white")
segment_3.goto(-40, 0)
```

**Lý do vị trí như vậy:**

- Mỗi turtle có kích thước 20x20 pixels
- Segment đầu tiên ở tọa độ (0, 0)
- Các segment tiếp theo lùi về trái 20 pixels để không chồng lên nhau


#### Cách 2: Sử dụng vòng lặp (tối ưu hơn)

```python
starting_positions = [(0, 0), (-20, 0), (-40, 0)]

for position in starting_positions:
    segment = Turtle("square")
    segment.color("white")
    segment.goto(position)
```

**Ưu điểm của cách 2:**

- Code ngắn gọn, dễ bảo trì
- Dễ mở rộng thêm nhiều segment
- Sử dụng **[[tuple]]** để lưu tọa độ (x, y)
- Áp dụng **[[for loop]]** để tạo nhiều đối tượng


### Code hoàn chỉnh

```python
from turtle import Screen, Turtle

screen = Screen()
screen.setup(width=600, height=600)
screen.bgcolor("black")
screen.title("My Snake Game")

starting_positions = [(0, 0), (-20, 0), (-40, 0)]

for position in starting_positions:
    segment = Turtle("square")
    segment.color("white")
    segment.goto(position)

screen.exitonclick()
```


### Kiến thức cần nhớ

- **Tuple (bộ giá trị)** - Cấu trúc dữ liệu lưu nhiều giá trị không thay đổi, viết trong dấu ngoặc đơn: `(x, y)`
- **goto() method** - Di chuyển turtle đến tọa độ cụ thể
- **Keyword arguments** - Sử dụng tên tham số khi gọi hàm để code rõ ràng hơn: `width=600`
- Mỗi turtle mặc định có kích thước 20x20 pixels
