

## Giới Thiệu Ngày 21 - Hoàn Thiện Snake Game

### Tổng quan buổi học

Ngày 21 tập trung vào việc hoàn thành game Snake với các khái niệm lập trình nâng cao:

- **Class Inheritance (Kế thừa class)** - Khái niệm quan trọng trong Object Oriented Programming
- **Slicing** - Kỹ thuật cắt list và dictionary trong Python
- **Hoàn thiện 4 tính năng cuối** của game Snake


### Khái niệm Class Inheritance

**Inheritance (Kế thừa)** là cơ chế cho phép một class có thể:

- Kế thừa attributes (thuộc tính) từ class khác
- Kế thừa methods (phương thức) từ class khác
- Mở rộng và tùy chỉnh chức năng của class cha
- Tạo code dễ bảo trì và tái sử dụng

**Lợi ích:**

- Giảm duplicate code
- Tạo cấu trúc phân cấp rõ ràng
- Dễ dàng modify và extend classes
- Tăng khả năng tái sử dụng code


### Slicing trong Python

**Slicing** là kỹ thuật cắt và truy xuất một phần của:

- Lists (danh sách)
- Tuples (bộ giá trị)
- Strings (chuỗi)
- Dictionaries (từ điển - Python 3.7+)

Cho phép lấy nhiều phần tử cùng lúc thay vì từng phần tử một.

### Các tính năng còn lại cần hoàn thiện

#### Bước 4: Phát hiện va chạm với thức ăn (Food Collision Detection)

- Tạo đối tượng Food xuất hiện ngẫu nhiên trên màn hình
- Detect khi đầu rắn chạm vào thức ăn
- Thức ăn mới xuất hiện ở vị trí ngẫu nhiên khác
- Rắn tăng chiều dài khi ăn được thức ăn


#### Bước 5: Tạo bảng điểm (Scoreboard)

- Hiển thị điểm số trên màn hình
- Cập nhật điểm mỗi khi rắn ăn được thức ăn
- Học cách viết text trong Turtle graphics
- Theo dõi điểm số trong suốt game


#### Bước 6: Phát hiện va chạm với tường (Wall Collision)

- Xác định giới hạn màn hình game
- Detect khi đầu rắn chạm vào tường
- Hiển thị "Game Over" khi va chạm
- Dừng game loop


#### Bước 7: Phát hiện va chạm với thân rắn (Tail Collision)

- Detect khi đầu rắn chạm vào bất kỳ phần nào của thân
- Càng dài rắn, càng dễ xảy ra va chạm
- Kết thúc game khi tự cắn mình
- Hiển thị "Game Over"
