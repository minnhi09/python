### TODO 1: Chọn từ ngẫu nhiên

**Yêu cầu:** Chọn ngẫu nhiên một từ từ `word_list` và gán vào biến `chosen_word`, sau đó in ra.

**Cách thực hiện:**

```python
import random

# Chọn ngẫu nhiên từ word_list
chosen_word = random.choice(word_list)
print(chosen_word)
```

**Giải thích:**

- Import [[Random Module]] để sử dụng hàm tạo số ngẫu nhiên
- Sử dụng `random.choice()` để chọn một phần tử ngẫu nhiên từ sequence (chuỗi/danh sách)
- Truyền `word_list` làm tham số
- Gán kết quả vào biến `chosen_word`
- In ra để kiểm tra

**Lưu ý về PEP 8:** Theo chuẩn Python, lệnh import nên đặt ở đầu file. PyCharm sẽ có gạch dưới màu vàng để nhắc nhở điều này.

### TODO 2: Nhận input từ người chơi

**Yêu cầu:** Yêu cầu người dùng đoán một chữ cái, gán vào biến `guess` và chuyển thành chữ thường.

**Cách thực hiện:**

```python
guess = input("Guess a letter: ").lower()
print(guess)  # In ra để kiểm tra
```

**Giải thích:**

- `input()` nhận dữ liệu từ người dùng
- `.lower()` chuyển đổi input thành chữ thường
- Lý do chuyển thành lowercase: Đảm bảo thống nhất vì `word_list` chứa toàn chữ thường, dù người dùng nhập chữ hoa hay thường đều được xử lý giống nhau

**Ví dụ:**

```
Người dùng nhập: A
Kết quả sau .lower(): a
```


### TODO 3: Kiểm tra chữ cái trong từ

**Yêu cầu:** Duyệt qua từng chữ cái trong `chosen_word`, so sánh với `guess`. In "Right" nếu trùng, "Wrong" nếu không trùng.

**Cách thực hiện:**

```python
for letter in chosen_word:
    if letter == guess:
        print("Right")
    else:
        print("Wrong")
```

**Giải thích từng bước:**

**Vòng lặp qua chuỗi:**

- `for letter in chosen_word:` - Duyệt qua từng ký tự trong chuỗi
- `letter` là biến tạm lưu từng ký tự (có thể đặt tên tùy ý)
- Python có thể loop qua string giống như loop qua list

**Điều kiện kiểm tra:**

- `if letter == guess:` - So sánh ký tự hiện tại với chữ cái người dùng đoán
- In "Right" nếu khớp
- In "Wrong" nếu không khớp
- Lưu ý: `if` và `else` phải thẳng hàng (cùng mức indentation)

**Ví dụ thực tế:**

```
Từ được chọn: "camel"
Người chơi đoán: "a"

Quá trình kiểm tra:
c == a? → Wrong
a == a? → Right
m == a? → Wrong
e == a? → Wrong
l == a? → Wrong
```
