### TODO 1: Tạo Placeholder

**Yêu cầu:** Tạo chuỗi placeholder với số dấu gạch dưới bằng số chữ cái trong `chosen_word`.

**Cách thực hiện:**

```python
# Tạo empty string
placeholder = ""

# Tính độ dài của từ
word_length = len(chosen_word)

# Tạo placeholder với vòng lặp
for position in range(word_length):
    placeholder += "_"

print(placeholder)
```

**Giải thích chi tiết:**

**Bước 1: Khởi tạo empty string**

```python
placeholder = ""
```

- Tạo một chuỗi rỗng để lưu trữ các dấu gạch dưới
- Empty string được tạo bằng cặp dấu ngoặc kép không có gì bên trong

**Bước 2: Tính độ dài từ**

```python
word_length = len(chosen_word)
```

- Sử dụng [[len() Function]] để đếm số ký tự trong chuỗi
- Ví dụ: `len("apple")` trả về `5`

**Bước 3: Tạo vòng lặp với range()**

```python
for position in range(word_length):
    placeholder += "_"
```

- `range(word_length)` tạo dãy số từ 0 đến `word_length - 1`
- Vòng lặp chạy đúng số lần bằng số chữ cái
- Mỗi lần lặp, thêm một dấu `_` vào placeholder
- `+=` là toán tử nối chuỗi (concatenation operator)

**Ví dụ thực tế:**

```python
chosen_word = "apple"  # 5 chữ cái
word_length = 5

# Vòng lặp chạy 5 lần:
# Lần 1: placeholder = "_"
# Lần 2: placeholder = "__"
# Lần 3: placeholder = "___"
# Lần 4: placeholder = "____"
# Lần 5: placeholder = "_____"
```


### TODO 2: Tạo Display với chữ cái đúng

**Yêu cầu:** Tạo biến `display` để hiển thị chữ cái đã đoán đúng ở đúng vị trí, còn lại là dấu gạch dưới.

**Cách thực hiện:**

```python
# Khởi tạo display rỗng
display = ""

# Loop qua từng chữ cái trong chosen_word
for letter in chosen_word:
    if letter == guess:
        display += letter
    else:
        display += "_"

print(display)
```

**Giải thích chi tiết:**

**Khởi tạo biến display:**

```python
display = ""
```

- Tạo chuỗi rỗng để xây dựng kết quả hiển thị

**Vòng lặp qua từng chữ cái:**

```python
for letter in chosen_word:
```

- Loop trực tiếp qua chuỗi (không cần `range()`)
- `letter` là biến tạm lưu từng ký tự

**Logic kiểm tra và thêm ký tự:**

```python
if letter == guess:
    display += letter  # Thêm chữ cái đúng
else:
    display += "_"     # Thêm underscore nếu chưa đoán
```

- So sánh từng chữ cái với chữ cái người dùng đoán
- Nếu khớp: Thêm chữ cái vào display
- Nếu không khớp: Thêm `_` vào display

**Ví dụ hoạt động:**

```python
chosen_word = "aardvark"
guess = "a"

# Quá trình loop:
# letter = "a" → a == a? Yes → display = "a"
# letter = "a" → a == a? Yes → display = "aa"
# letter = "r" → r == a? No  → display = "aa_"
# letter = "d" → d == a? No  → display = "aa__"
# letter = "v" → v == a? No  → display = "aa___"
# letter = "a" → a == a? Yes → display = "aa___a"
# letter = "r" → r == a? No  → display = "aa___a_"
# letter = "k" → k == a? No  → display = "aa___a__"

# Kết quả cuối: "aa___a__"
```

### Lưu ý quan trọng

**Về range():**

- `range(5)` tạo dãy: 0, 1, 2, 3, 4 (không bao gồm 5)
- `range(1, 6)` tạo dãy: 1, 2, 3, 4, 5 (không bao gồm 6)
- Số lần lặp = `end - start`

**Về empty string:**

- Khởi tạo: `variable = ""`
- Không có khoảng trắng giữa hai dấu ngoặc kép
- Dùng để xây dựng chuỗi dần dần

**Debug tips:**

- In `placeholder` để xem số underscore có đúng không
- In `display` để kiểm tra logic thay thế
- In `letter` trong vòng lặp để theo dõi quá trình
