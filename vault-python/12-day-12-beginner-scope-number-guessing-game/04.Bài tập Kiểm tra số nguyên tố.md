



### Yêu cầu

Viết hàm `is_prime()` để kiểm tra xem một số có phải là số nguyên tố hay không.

**Định nghĩa số nguyên tố (Prime Number):**

- Số chỉ chia hết cho 1 và chính nó
- Ví dụ: 7 là số nguyên tố (chỉ chia hết cho 1 và 7)
- Ví dụ: 4 không phải số nguyên tố (chia hết cho 1, 2, 4)

**Lưu ý đặc biệt:**

- Số 2 là số nguyên tố
- Số 1 không phải số nguyên tố


### Giải pháp

```python
def is_prime(n):
    # Trường hợp đặc biệt
    if n <= 1:
        return False
    if n == 2:
        return True
    if n % 2 == 0:
        return False
    
    # Kiểm tra các số lẻ từ 3 đến căn bậc hai của n
    for i in range(3, int(n**0.5) + 1, 2):
        if n % i == 0:
            return False
    
    return True

# Test
print(is_prime(73))  # True
print(is_prime(75))  # False
```


### Giải thích logic

**Bước 1: Xử lý trường hợp đặc biệt**

- Nếu `n <= 1`: trả về `False` (1 và số âm không phải số nguyên tố)
- Nếu `n == 2`: trả về `True` (2 là số nguyên tố duy nhất chẵn)
- Nếu `n % 2 == 0`: trả về `False` (các số chẵn khác không phải số nguyên tố)

**Bước 2: Kiểm tra ước số**

- Chỉ cần kiểm tra đến căn bậc hai của n: `int(n**0.5) + 1`
- Ví dụ với n=73: chỉ cần kiểm tra đến 8 (vì 9² = 81 > 73)
- Chỉ kiểm tra số lẻ: `range(3, ..., 2)` (bỏ qua số chẵn)
- Nếu tìm thấy ước số → không phải số nguyên tố

**Bước 3: Kết luận**

- Nếu không tìm thấy ước số nào → là số nguyên tố


### Tối ưu hóa

**Tại sao chỉ kiểm tra đến căn bậc hai?**

Nếu n có ước số a (với a > √n), thì phải tồn tại ước số b = n/a (với b < √n). Vậy nên chỉ cần kiểm tra đến √n là đủ.

**Ví dụ với n=36:**

- √36 = 6
- Các cặp ước: (1,36), (2,18), (3,12), (4,9), (6,6)
- Chỉ cần kiểm tra đến 6 để tìm ra các ước


### Test cases

```python
# Test các trường hợp
print(is_prime(1))   # False - 1 không phải số nguyên tố
print(is_prime(2))   # True - 2 là số nguyên tố
print(is_prime(4))   # False - chia hết cho 2
print(is_prime(7))   # True - số nguyên tố
print(is_prime(73))  # True - số nguyên tố
print(is_prime(75))  # False - chia hết cho 3, 5
print(is_prime(97))  # True - số nguyên tố
```
