

## Module Turtle Graphics - Làm Việc Với Đồ Họa

### Giới Thiệu Module Turtle

Module Turtle (Rùa) trong Python là công cụ cho phép vẽ đồ họa lên màn hình. Hình dung module này như một chú rùa nhỏ mang bút trên lưng, và mọi nơi nó đi qua sẽ để lại một vệt nét vẽ.

### Cách Import và Thiết Lập

Có hai cách import module turtle:

```python
from turtle import Turtle, Screen

# Tạo đối tượng turtle
timmy_the_turtle = Turtle()

# Tạo đối tượng screen để giữ cửa sổ mở
screen = Screen()
screen.exitonclick()  # Đặt ở cuối code
```

**Lưu ý quan trọng:** Dòng `screen.exitonclick()` phải đặt ở cuối file để cửa sổ chỉ đóng khi người dùng click vào.

### Thay Đổi Hình Dạng Turtle

Sử dụng phương thức `shape()` để thay đổi hình dạng:

```python
timmy_the_turtle.shape("turtle")
```

**Các hình dạng có sẵn:**

- arrow (mũi tên - mặc định)
- turtle (rùa)
- circle (hình tròn)
- square (hình vuông)
- triangle (tam giác)
- classic (cổ điển)


### Thay Đổi Màu Sắc

Sử dụng phương thức `color()` để đổi màu:

```python
timmy_the_turtle.color("red")
```

Module turtle sử dụng Tk color specifications (đặc tả màu Tk), bao gồm hàng trăm tên màu như "red", "green", "coral", "deep sky blue", v.v.

**Nguồn tham khảo màu:** Module turtle dựa trên tkinter (Tk interface) - công cụ tạo giao diện đồ họa người dùng (GUI) trong Python.

### Các Lệnh Di Chuyển Cơ Bản

**Di chuyển tuyến tính:**

```python
timmy_the_turtle.forward(100)   # Tiến 100 bước
timmy_the_turtle.backward(100)  # Lùi 100 bước
```

**Xoay hướng:**

```python
timmy_the_turtle.right(90)   # Xoay phải 90 độ
timmy_the_turtle.left(90)    # Xoay trái 90 độ
```

**Góc xoay:** Nhận giá trị từ 0 đến 360 độ. Ví dụ, turtle đang hướng Đông và muốn quay sang Nam thì cần xoay phải 90 độ.

### Khái Niệm GUI (Graphical User Interface)

**GUI là gì?**

- Giao diện đồ họa người dùng cho phép tương tác bằng chuột và hình ảnh
- Khác với text interface (giao diện văn bản) như MS-DOS hay Python console
- Ví dụ lịch sử: Apple Lisa 1 - máy tính đầu tiên có GUI

**Module tkinter:**

- Viết tắt của "Tk interface"
- Công cụ Python để tạo GUI
- Module turtle sử dụng tkinter ở phía dưới để tạo đồ họa
