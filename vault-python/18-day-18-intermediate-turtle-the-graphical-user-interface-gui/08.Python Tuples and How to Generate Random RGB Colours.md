

## Python Tuples và Màu RGB Ngẫu Nhiên

### Giới Thiệu Python Tuples

**Tuple là gì?**

Tuple là một kiểu dữ liệu (data type) trong Python, có cú pháp sử dụng dấu ngoặc tròn:

```python
my_tuple = (1, 3, 8)
```

**Cách truy cập phần tử:**

```python
my_tuple = (1, 3, 8)
print(my_tuple[0])  # Output: 1
print(my_tuple[1])  # Output: 3
print(my_tuple[2])  # Output: 8
```

Giống như list, tuple cũng sử dụng index bắt đầu từ 0 và dùng dấu ngoặc vuông `[]` để truy cập.

### So Sánh Tuple và List

**Điểm giống nhau:**

- Cả hai đều lưu trữ nhiều phần tử
- Các phần tử có thứ tự (ordered)
- Truy cập bằng index

**Điểm khác nhau quan trọng:**

```python
# List - Có thể thay đổi (mutable)
my_list = [1, 3, 8]
my_list[2] = 12  # ✅ Hoạt động bình thường
print(my_list)   # [1, 3, 12]

# Tuple - Không thể thay đổi (immutable)
my_tuple = (1, 3, 8)
my_tuple[2] = 12  # ❌ TypeError: tuple does not support item assignment
```

**Immutable (Bất biến):**

- Tuple không thể thay đổi sau khi tạo
- Không thể gán lại giá trị
- Không thể xóa phần tử
- Giống như "khắc trên đá" - carved in stone


### Khi Nào Nên Dùng Tuple?

**Trường hợp sử dụng Tuple:**

- Dữ liệu không nên thay đổi (ví dụ: màu sắc cố định)
- Tránh việc vô tình thay đổi dữ liệu quan trọng
- Bảng màu cho website
- Tọa độ cố định
- Cấu hình hệ thống không đổi

**Chuyển đổi giữa Tuple và List:**

```python
my_tuple = (1, 3, 8)
my_list = list(my_tuple)  # Chuyển tuple thành list
print(my_list)            # [1, 3, 8]
```


### RGB Colors - Hệ Màu RGB

**Khái niệm RGB:**

RGB là viết tắt của Red (Đỏ), Green (Xanh lá), Blue (Xanh dương) - ba màu cơ bản có thể tạo ra mọi màu sắc.

**Cách hoạt động:**

- Mỗi màu có giá trị từ 0 đến 255
- $0, 0, 0$ = Đen (không có màu nào)
- $255, 255, 255$ = Trắng (đầy đủ cả ba màu)
- Kết hợp khác nhau tạo ra vô số màu

**Ví dụ màu RGB:**

```python
black = (0, 0, 0)
white = (255, 255, 255)
red = (255, 0, 0)
green = (0, 255, 0)
blue = (0, 0, 255)
yellow = (255, 255, 0)  # Đỏ + Xanh lá
```


### RGB trong Turtle Graphics

**Cấu hình Color Mode:**

Turtle mặc định sử dụng color mode từ 0.0 đến 1.0, cần thay đổi sang 0-255:

```python
import turtle as t

t.colormode(255)  # Thay đổi sang chế độ 0-255
```

**⚠️ Lưu ý quan trọng:** Phải gọi `colormode(255)` trên module turtle, không phải trên turtle object.

**Sử dụng RGB color:**

```python
import turtle as t

t.colormode(255)
tim = t.Turtle()

# Sử dụng tuple màu RGB
tim.color((255, 0, 0))  # Màu đỏ
```


### Tạo Màu RGB Ngẫu Nhiên

**Function tạo màu ngẫu nhiên:**

```python
import random

def random_color():
    r = random.randint(0, 255)
    g = random.randint(0, 255)
    b = random.randint(0, 255)
    color = (r, g, b)
    return color
```

**Giải thích:**

- `random.randint(0, 255)`: Tạo số nguyên ngẫu nhiên từ 0 đến 255
- Tạo ba số ngẫu nhiên cho R, G, B
- Gói vào tuple `(r, g, b)`
- Return tuple màu ngẫu nhiên

**Version rút gọn:**

```python
import random

def random_color():
    r = random.randint(0, 255)
    g = random.randint(0, 255)
    b = random.randint(0, 255)
    return (r, g, b)
```


### Code Hoàn Chỉnh - Random Walk Với RGB Random

```python
import turtle as t
import random

tim = t.Turtle()
t.colormode(255)

directions = [0, 90, 180, 270]
tim.pensize(15)
tim.speed("fastest")

def random_color():
    r = random.randint(0, 255)
    g = random.randint(0, 255)
    b = random.randint(0, 255)
    return (r, g, b)

for _ in range(200):
    tim.color(random_color())
    tim.forward(30)
    tim.setheading(random.choice(directions))

screen = t.Screen()
screen.exitonclick()
```


### So Sánh: Color List vs RGB Random

**Cách cũ - Chọn từ danh sách:**

```python
colors = ["red", "blue", "green", "yellow"]
tim.color(random.choice(colors))  # Giới hạn ở các màu có sẵn
```

**Cách mới - RGB ngẫu nhiên:**

```python
tim.color(random_color())  # Vô số màu có thể
```

**Ưu điểm RGB random:**

- Không giới hạn số lượng màu
- Mỗi lần chạy tạo màu hoàn toàn khác nhau
- Linh hoạt và đa dạng hơn

### Ứng Dụng Thực Tế

**Random Walk với màu RGB:**

Kết hợp random walk từ bài trước với RGB random color tạo ra:

- Tranh nghệ thuật độc đáo mỗi lần chạy
- Pattern màu sắc không lặp lại
- Hiệu ứng thị giác phong phú

**Kết quả:** Mỗi bước trong random walk có màu hoàn toàn ngẫu nhiên, tạo nên tác phẩm nghệ thuật đa dạng và sinh động.

### Ghi Chú Thêm

**Về Tuple:**

- Dùng `()` để tạo tuple
- Immutable - không thể sửa đổi
- Phù hợp cho dữ liệu cố định
- Hiệu suất tốt hơn list một chút

**Về RGB:**

- 3 màu cơ bản: Red, Green, Blue
- Mỗi màu từ 0-255 (256 giá trị)
- Tổng cộng: $256 \times 256 \times 256 = 16,777,216$ màu có thể

**Về Color Mode:**

- Turtle mặc định: 0.0 - 1.0
- Thay đổi sang 0-255 dễ dùng hơn
- Phải set trên module, không phải object
