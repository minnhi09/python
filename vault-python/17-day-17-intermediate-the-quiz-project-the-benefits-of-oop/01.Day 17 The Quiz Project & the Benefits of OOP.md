

## Giới thiệu Day 17 - Tạo Classes riêng

### Chào mừng đến Day 17

Hôm nay tập trung vào việc học cách **tạo các classes** (lớp) mà chúng ta đã sử dụng trong các bài học ngày hôm qua.

### Dự án: Quiz Game (Trò chơi Đố vui)

**Mục tiêu:**
Xây dựng một trò chơi quiz dạng True/False (Đúng/Sai) hoàn chỉnh sử dụng Object-Oriented Programming.

**Tính năng chính:**

- Tạo quiz với câu hỏi True/False
- Chơi quiz và trả lời câu hỏi
- Theo dõi điểm số (score tracking)

**Ví dụ gameplay:**

```
Question 1: A slug's blood is green.
True or False? True
✓ Correct!
```


### Phương pháp xây dựng

**Sử dụng OOP toàn diện:**

- Tự tạo classes riêng (không dùng code có sẵn)
- Tạo objects từ classes đó
- Sử dụng objects để xây dựng game logic

**Khác với Day 16:**

- Day 16: Sử dụng classes có sẵn (CoffeeMaker, MoneyMachine)
- Day 17: **Tự tạo classes từ đầu**


### Nội dung học

**Kiến thức mới:**

- Cách tạo custom classes (classes tùy chỉnh)
- Cách khai báo (declare) classes
- Khái niệm initialization (khởi tạo)
- Từ khóa `self` và cách sử dụng

**Quy trình:**

1. Thiết kế classes cho quiz game
2. Tạo classes với attributes và methods
3. Khởi tạo objects từ classes
4. Kết hợp objects để xây dựng game logic

### Chuẩn bị

**Trải nghiệm trước:**

- Link đến phiên bản hoàn chỉnh của Quiz Game có trong course resources
- Chơi thử để hiểu cách game hoạt động
- Nắm rõ yêu cầu trước khi bắt đầu code

**Bước tiếp theo:**
Bài học sau sẽ đi sâu vào cách tạo custom classes từ đầu.

### Tầm quan trọng của Day 17

**Chuyển từ User sang Creator:**

- Trước đây: Sử dụng classes người khác tạo
- Từ giờ: Tự thiết kế và tạo classes riêng
- Là bước quan trọng để thành thạo OOP

**Kỹ năng đạt được:**

- Tư duy thiết kế classes
- Hiểu cấu trúc bên trong classes
- Tự chủ trong việc tổ chức code
- Nền tảng để xây dựng projects phức tạp


### Điểm nhấn Quiz Game

**Đặc điểm:**

- Dạng câu hỏi: True/False đơn giản
- Tracking score tự động
- Feedback ngay lập tức sau mỗi câu trả lời
- Có thể mở rộng với nhiều câu hỏi hơn

**Ứng dụng thực tế:**
Học cách tạo classes qua Quiz Game là nền tảng để:

- Tạo các ứng dụng game phức tạp hơn
- Xây dựng hệ thống quản lý data
- Thiết kế software architecture tốt hơn
