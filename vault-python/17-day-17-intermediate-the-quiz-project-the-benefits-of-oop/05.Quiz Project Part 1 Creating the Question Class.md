### Bước 1: Tạo Question Model

**Phân tích yêu cầu:**

Một Question object cần có gì?

**Attributes cần thiết:**

- `text` - Nội dung câu hỏi
- `answer` - Đáp án đúng (True/False)

**Initialization:**
Cả hai attributes phải được khởi tạo khi tạo Question object mới.

### Thiết kế Question Class

**Ví dụ mong muốn:**

```python
# Câu hỏi: "2 + 3 = 5"
# Đáp án: True

new_q = Question("2 + 3 = 5", True)
```

**Flow hoạt động:**

```
Question("2 + 3 = 5", True)
    ↓
__init__ nhận parameters
    ↓
self.text = "2 + 3 = 5"
self.answer = True
    ↓
new_q object được tạo với attributes đã khởi tạo
```

**Bước 3: Thêm attributes**

```python
class Question:
    def __init__(self):
        self.text = ""
        self.answer = ""
```

**Lưu ý:**

- `self.text` = Attribute lưu câu hỏi
- `self.answer` = Attribute lưu đáp án
- Hiện tại yellowed out vì chưa nhận giá trị từ bên ngoài

**Bước 4: Thêm parameters**

```python
class Question:
    def __init__(self, q_text, q_answer):
        self.text = q_text
        self.answer = q_answer
```

**Giải thích chi tiết:**


| Element | Ý nghĩa |
| :-- | :-- |
| `q_text` | Parameter nhận câu hỏi từ bên ngoài |
| `q_answer` | Parameter nhận đáp án từ bên ngoài |
| `self.text` | Attribute lưu câu hỏi trong object |
| `self.answer` | Attribute lưu đáp án trong object |

**Mapping:**

- `q_text` → `self.text`
- `q_answer` → `self.answer`

### Naming Conventions

**Parameters vs Attributes:**

```python
def __init__(self, q_text, q_answer):
    self.text = q_text      # Parameter ≠ Attribute name
    self.answer = q_answer  # OK - tên khác nhau
```

**Có thể đặt tên giống nhau:**

```python
def __init__(self, text, answer):
    self.text = text        # Parameter = Attribute name
    self.answer = answer    # Cũng OK
```

**Lưu ý:**

- Tên parameters có thể tùy ý
- Quan trọng là mapping đúng giá trị
- Convention: Thường đặt tên khác để phân biệt rõ


### Ý nghĩa của từng dòng code

```python
class Question:
```

Khai báo class tên Question (PascalCase)

```python
def __init__(self, q_text, q_answer):
```

- Constructor tự động gọi khi tạo object
- `self` = object đang được tạo
- `q_text`, `q_answer` = data từ bên ngoài

```python
self.text = q_text
```

Gán giá trị từ parameter `q_text` vào attribute `self.text`

```python
self.answer = q_answer
```

Gán giá trị từ parameter `q_answer` vào attribute `self.answer`
