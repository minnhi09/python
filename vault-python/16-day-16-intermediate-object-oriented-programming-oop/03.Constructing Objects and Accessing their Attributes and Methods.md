

## Triển khai OOP trong Python

### Class Blueprint và Object

**Class (Lớp) - Blueprint:**

- Là bản thiết kế chi tiết của một đối tượng
- Quy định những gì object có (attributes) và làm được (methods)
- Ví dụ: Blueprint của một chiếc xe xác định: color, wheels, mileage, fuel + các chức năng drive(), stop(), break()

**Object (Đối tượng):**

- Là thực thể cụ thể được tạo từ class blueprint
- Là thứ chúng ta thực sự sử dụng trong code
- Có thể tạo nhiều objects từ cùng một class


### Cú pháp Tạo Object trong Python

```python
# Cú pháp cơ bản
car = CarBlueprint()
```

**Quy tắc đặt tên:**

- **Class name**: Viết theo Pascal Case (ví dụ: `CarBlueprint`, `MyClass`)
- **Object/variable name**: Viết thường, ngăn cách bằng underscore (ví dụ: `my_car`, `timmy`)

**Cách hoạt động:**

- Tên object = Tên Class + dấu ngoặc đơn `()`
- Dấu ngoặc đơn kích hoạt việc xây dựng (construct) object từ blueprint


### Làm việc với Module và Class

**Import module:**

```python
# Cách 1: Import cả module
import turtle
timmy = turtle.Turtle()

# Cách 2: Import class cụ thể (đơn giản hơn)
from turtle import Turtle
timmy = Turtle()
```

**Phân biệt trong code:**

- `turtle` (chữ thường) = tên module
- `Turtle` (chữ hoa đầu) = tên Class (ký hiệu C trong PyCharm)
- Dấu ngoặc đơn `()` = khởi tạo/construct object


### Attributes (Thuộc tính)

**Khái niệm:**
Attributes là các biến gắn liền với object, đại diện cho dữ liệu mà object lưu trữ (ví dụ: speed, fuel của xe).

**Cú pháp truy cập:**

```python
# object.attribute
print(car.speed)
print(my_screen.canvasheight)
```

**Dot notation (dấu chấm):**

- Dùng dấu `.` để truy cập attribute từ object
- Cấu trúc: `object_name.attribute_name`


### Methods (Phương thức)

**Khái niệm:**
Methods là các functions gắn liền với object, đại diện cho những hành động mà object có thể thực hiện.

**Cú pháp gọi method:**

```python
# object.method()
car.stop()
my_screen.exitonclick()
timmy.forward(100)
```

**Đặc điểm:**

- Sử dụng dot notation giống attributes
- Có dấu ngoặc đơn `()` vì là function
- Có thể nhận tham số (parameters)


### Thực hành với Turtle Graphics

**Thiết lập project:**

```python
from turtle import Turtle, Screen

# Tạo turtle object
timmy = Turtle()
print(timmy)  # In ra: <turtle.Turtle object at 0x...>

# Tạo screen object
my_screen = Screen()
print(my_screen.canvasheight)  # In ra: 300
```

**Các methods phổ biến:**

```python
# Thay đổi hình dạng
timmy.shape("turtle")

# Thay đổi màu sắc
timmy.color("coral")

# Di chuyển
timmy.forward(100)  # Di chuyển tiến 100 bước

# Giữ màn hình đến khi click
my_screen.exitonclick()
```

**Đọc Documentation:**

- Turtle graphics có documentation đầy đủ
- Liệt kê tất cả methods có thể sử dụng: move, draw, position, color, reset...
- Mỗi method có hướng dẫn cách sử dụng và parameters cần truyền vào


### Ví dụ Code Hoàn chỉnh

```python
from turtle import Turtle, Screen

# Tạo turtle object
timmy = Turtle()

# Tuỳ chỉnh turtle
timmy.shape("turtle")
timmy.color("coral")

# Di chuyển turtle
timmy.forward(100)

# Tạo screen và giữ cho đến khi click
my_screen = Screen()
my_screen.exitonclick()
```


### Module Khác

**Tạo module riêng:**

```python
# File: another_module.py
another_variable = 12

# File: main.py
import another_module
print(another_module.another_variable)  # Output: 12
```


### Tổng kết

**Quy trình làm việc với Objects:**

1. Import module hoặc class
2. Tạo object từ class blueprint: `object_name = ClassName()`
3. Truy cập attributes: `object.attribute`
4. Gọi methods: `object.method(parameters)`

**Lợi ích:**

- Sử dụng code của người khác đã viết sẵn
- Không cần viết lại chức năng phức tạp
- Chỉ cần đọc documentation và gọi methods phù hợp
- Có thể mở rộng với nhiều external libraries/packages khác

**Bước tiếp theo:**
Học cách tìm và sử dụng thêm nhiều external libraries/packages ngoài turtle để mở rộng khả năng của Python.
