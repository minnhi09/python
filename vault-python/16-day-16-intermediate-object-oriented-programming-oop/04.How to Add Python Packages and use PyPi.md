

## Sử dụng Packages trong Python

### Package là gì?

**Sự khác biệt giữa Module và Package:**

**Module:**

- Mỗi file Python (.py) trong project là một module
- Là một file code đơn lẻ

**Package:**

- Là một bộ sưu tập lớn code do người khác viết
- Bao gồm nhiều files được đóng gói lại với nhau
- Phục vụ một mục đích hoặc chức năng cụ thể


### Vấn đề cần giải quyết

**Ví dụ: Tạo bảng Pokemon**

Muốn tạo bảng hiển thị Pokemon và Type của chúng trong console (ASCII format):

```
| Pokemon | Type     |
|---------|----------|
| Pikachu | Electric |
| Squirtle| Water    |
```

**Thách thức khi tự làm:**

```python
# Code thủ công - khó và phức tạp
print("| Pokemon | Type |")
print("|---------|------|")
# ... phải format thủ công từng dòng
```

- Khó visualize kết quả
- Phải format và căn chỉnh thủ công
- Tốn thời gian và dễ sai

**Giải pháp:** Sử dụng package có sẵn như `PrettyTable`

### PyPI - Python Package Index

**Giới thiệu:**

- Website: [pypi.org](https://pypi.org)
- Là kho lưu trữ các packages/libraries cho Python
- Do cộng đồng Python phát triển và chia sẻ
- Có thể xem source code và documentation

**Cách tìm package:**

1. Truy cập PyPI
2. Tìm kiếm package theo nhu cầu (ví dụ: "prettytable")
3. Đọc documentation để hiểu cách sử dụng
4. Cài đặt vào project

### Cài đặt Package trong PyCharm

**Bước 1: Mở Preferences**

- **Windows**: File → Preferences
- **Mac**: PyCharm → Preferences

**Bước 2: Vào Project Settings**

- Click vào tên project của bạn (ví dụ: `day-16-start`)
- Chọn **Project Interpreter**

**Bước 3: Cài đặt Package**

- Click nút dấu cộng `+`
- Tìm kiếm package (ví dụ: `PrettyTable`)
- Chọn package và click **Install**
- Click **OK** khi hoàn tất


### Sử dụng Package sau khi cài đặt

**Import package:**

```python
import prettytable
```

**Xem source code:**

- Right-click vào tên package
- Chọn **Go to Implementation**
- Sẽ thấy toàn bộ code Python của package

**Lưu ý quan trọng:**

- Không cần hiểu chi tiết cách package hoạt động bên trong
- Chỉ cần đọc documentation để biết cách sử dụng
- Tập trung vào implement vào project của mình


### So sánh: Built-in vs External Packages

**Built-in packages (có sẵn):**

- Ví dụ: `turtle`
- Không cần cài đặt
- Đi kèm với mỗi bản Python

**External packages (từ PyPI):**

- Ví dụ: `prettytable`
- Cần cài đặt qua PyPI
- Được cộng đồng phát triển


### Documentation

**Tìm documentation:**

- Trên trang PyPI của package
- Click vào **Project Links**
- Thường có tutorials và examples

**PrettyTable documentation:**

- Có hướng dẫn tutorial
- Ít phức tạp hơn Turtle graphics
- Đủ chức năng cho mục đích tạo bảng ASCII


### Quy trình làm việc với External Packages

1. **Xác định nhu cầu** - Cần chức năng gì?
2. **Tìm kiếm trên PyPI** - Package nào phù hợp?
3. **Đọc documentation** - Hiểu cách sử dụng
4. **Cài đặt package** - Qua PyCharm Preferences
5. **Import và sử dụng** - Tạo objects, gọi methods
6. **Tham khảo documentation** - Khi cần tính năng cụ thể

### Ví dụ thực hành

**Bước tiếp theo:** Trong bài học sau, sẽ thực hành:

- Tạo PrettyTable object
- Sử dụng attributes và methods
- Khám phá documentation để thêm tính năng


### Lợi ích của việc sử dụng Packages

- Tiết kiệm thời gian - Không phải viết lại code phức tạp
- Đã được test - Code đã được cộng đồng kiểm tra
- Tập trung vào logic - Không cần lo chi tiết implementation
- Mở rộng khả năng - Làm được nhiều việc hơn với ít code hơn
- Cộng đồng hỗ trợ - Có documentation và examples
