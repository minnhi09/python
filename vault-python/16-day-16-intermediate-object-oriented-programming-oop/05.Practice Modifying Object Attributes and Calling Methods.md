

## Thực hành với PrettyTable Object

### Tạo PrettyTable Object

**Bước đầu tiên: Import class**

```python
from prettytable import PrettyTable
```

**Cú pháp tạo object:**

```python
table = PrettyTable()
```

- `table` = tên object (có thể đặt tên tùy ý)
- `PrettyTable()` = class blueprint với Pascal Case
- Dấu ngoặc đơn `()` = construct/khởi tạo object

**Kiểm tra object:**

```python
print(table)
# Output: Bảng trống với các ký tự ASCII
```


### Thêm Dữ liệu với Method add_column()

**Cú pháp method:**

```python
table.add_column(field_name, data_list)
```

**Parameters (tham số):**

- `field_name` (string): Tên cột (hiển thị ở header)
- `data_list` (list): Danh sách dữ liệu cho cột đó

**Ví dụ thực tế: Bảng Pokemon**

```python
from prettytable import PrettyTable

# Tạo table object
table = PrettyTable()

# Thêm cột đầu tiên - Pokemon name
table.add_column("Pokemon Name", ["Pikachu", "Squirtle", "Charmander"])

# Thêm cột thứ hai - Type
table.add_column("Type", ["Electric", "Water", "Fire"])

# In bảng
print(table)
```

**Kết quả:**

```
+--------------+----------+
| Pokemon Name |   Type   |
+--------------+----------+
|   Pikachu    | Electric |
|   Squirtle   |  Water   |
|  Charmander  |   Fire   |
+--------------+----------+
```

**Lưu ý quan trọng:**

- Dữ liệu trong hai cột phải theo đúng thứ tự tương ứng
- Pikachu → Electric, Squirtle → Water, Charmander → Fire
- Số phần tử trong mỗi list phải bằng nhau


### Thay đổi Attributes để Tùy chỉnh

**Khái niệm:**
Attributes kiểm soát giao diện và cách hiển thị của table (khác với methods là hành động).

**Attribute phổ biến: align (căn chỉnh)**

**Các giá trị:**

- `"l"` = Left-aligned (căn trái)
- `"c"` = Center-aligned (căn giữa - mặc định)
- `"r"` = Right-aligned (căn phải)

**Cách sử dụng:**

```python
# Kiểm tra giá trị hiện tại
print(table.align)
# Output: {'Pokemon Name': 'c', 'Type': 'c'}

# Thay đổi sang căn trái
table.align = "l"

# In bảng với căn trái
print(table)
```

**Kết quả sau khi căn trái:**

```
+--------------+----------+
| Pokemon Name | Type     |
+--------------+----------+
| Pikachu      | Electric |
| Squirtle     | Water    |
| Charmander   | Fire     |
+--------------+----------+
```


### So sánh Methods và Attributes

**Methods (Phương thức):**

- Là functions gắn với object
- Thực hiện hành động/chức năng
- Có dấu ngoặc đơn `()`
- Ký hiệu: `M` trong PyCharm
- Ví dụ: `table.add_column()`, `table.add_row()`

**Attributes (Thuộc tính):**

- Là variables gắn với object
- Lưu trữ dữ liệu/trạng thái
- Không có dấu ngoặc đơn
- Ký hiệu: `F` (field) trong PyCharm
- Ví dụ: `table.align`, `table.border`

**Cú pháp:**

```python
# Method - gọi để thực hiện hành động
table.add_column("Name", ["Data"])

# Attribute - đọc giá trị
print(table.align)

# Attribute - thay đổi giá trị
table.align = "l"
```


### Code Hoàn chỉnh

```python
from prettytable import PrettyTable

# Tạo table object
table = PrettyTable()

# Thêm dữ liệu
table.add_column("Pokemon Name", ["Pikachu", "Squirtle", "Charmander"])
table.add_column("Type", ["Electric", "Water", "Fire"])

# Tùy chỉnh giao diện
table.align = "l"  # Căn trái

# In bảng
print(table)
```


### Các Attributes khác để tùy chỉnh

**Theo documentation của PrettyTable:**

- `border` - Hiển thị/ẩn viền
- `header` - Hiển thị/ẩn header
- `padding_width` - Khoảng cách padding
- `sortby` - Sắp xếp theo cột
- Và nhiều options khác...
